# slot-machine
Slot machine simulation html5 web-app
## Demo
http://gdainti.com

## Contols
Click SPIN to play (enter and spacebar keys work as well)

## Development Setup

- Install Node.js and npm

- Install the Angular CLI globally:

```
npm install -g @angular/cli
```

- Install dependencies:

```
npm i
```

- Run the application:

```
ng serve
```

- Run unit tests:

```
ng test
```
